# `fn-html`

- Useful html functions

## Install

```bash
npm install fn-html
```

## Usage

- see [USAGE.md](USAGE.md)

### Quick Example

```js
const html = require("fn-html");

console.log(html.p("Hello World"));

// print <P>Hello World</P>
```
